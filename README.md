# flutter-upskilling project

## To test my understanding of flutter, I created a flutter quiz application which:
 - Allow users start the quiz.
 - Select an answer from a list of possible answers.
 - Get graded results at the end.
 - Have the options to either restart the quiz or go back to the home screen.

## The app demostrates an understanding to the following flutter concepts:
 - Stateless and Stateful Widgets.
 - Uplifting the state
 - Classes and Objects.
 - Shared custom widgets.
 - Data models, Enums, Helper methods.

## To run the project:
 - Run >> `flutter pub get`
 - Run emulator: preferrably android emulator for this initial change as the design might be a litle off on browsers and other platforms.
   >> If on vs code: navigate to `view > command palette > flutter: launch emulator > choose preferred emulator`
 - Run >> `flutter run`
